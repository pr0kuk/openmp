# Lab Work
This repository contains solutions to problems for the MIPT MPI course (2022). <br>
All tasks were completed by the student of MIPT group B01-908 Alexey Shcherbakov.

# Requirements
*   Open MPI 4.1.2 or newer
*   GCC 9.4.0. or newer
# Build
*       cmake .
*       make
# Run
### OpenMP version
*       ./openmp
### MPI version
*       bash mpi.sh
